# kratke

> Short with numbers


## Install

```
$ npm install --save kratke
```


## Usage

```js
var kratke = require('kratke');

var name = 'http://www.thelongestdomainnameintheworldandthensomeandthensomemoreandmore.com/';

var code = kratke(name);

console.log(code);
```

## API

### kratke(name)

#### name

Type: `string`
